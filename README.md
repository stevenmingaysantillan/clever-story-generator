Description:

An interactive word-game utility inspired by traditional "Mad Libs." It prompts users for various parts of speech, activities, and emotional exclamations, then seamlessly weaves them into a pre-constructed narrative template to generate highly unpredictable and humorous stories.

Key Features
Expanded Vocabulary Prompts: Features specialized input prompts like "favorite thing to do" and "thing you hated the most" to drive highly personalized, funny outcomes.

String Formatting: Employs Python f-strings to dynamically inject user inputs into the text block.

Text Normalization: Uses methods like .capitalize() to ensure proper grammatical capitalization at the start of dialogue sentences.
